# 📊 Task 01 - Population Distribution Visualization

## 🎯 Objective
To visualize the distribution of population across countries using histogram and bar chart.

## 🛠️ Tools Used
- Python  
- Pandas  
- Matplotlib  
- Seaborn  
- Google Colab  

## 📌 Steps Performed
1. Loaded the dataset  
2. Checked column names and structure  
3. Cleaned the dataset (handled missing values)  
4. Selected population data for the year 2022  
5. Created histogram to show population distribution  
6. Created bar chart for top 10 most populated countries  

## 📁 Dataset
World Population Dataset (CSV file)
